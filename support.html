🔐 登录相关问题

  • • 登录失败
    请确认账号、密码或短信验证码是否正确，并检查当前网络连接（尝试切换 Wi-Fi / 蜂窝网络）。
  • • 无法接收验证码
    请检查手机号填写是否正确，查看短信拦截设置（骚扰拦截/黑名单），稍等 1-2 分钟后重新获取。
  • • Face ID 登录异常
    请确认系统设置中已为「谨小智」开启 Face ID 权限，并且当前设备支持面容识别。

🔔 消息与数据同步

  • • 消息未提醒或延迟
    检查 iOS 系统通知权限是否开启，以及 App 内「消息通知」开关是否处于启用状态。
  • • 数据未同步
    请下拉首页刷新数据，并确认当前网络可用（避免使用代理或 VPN）。
